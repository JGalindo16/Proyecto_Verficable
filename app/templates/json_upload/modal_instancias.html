📚 Subir Instancias de Curso

Sube el archivo 4-instancias_cursos.json para registrar cuándo se dicta cada curso.

{
          {
            año: 2024,
            semestre: 1,
            instancias: [
              {
                curso_id: 101,
                id: 1
              }
            ]
          }
  }
Validaciones:
  • ✔️ curso_id debe existir previamente en la base de datos.
  • ✔️ anio debe ser un número entero.
  • ✔️ semestre debe ser texto válido (e.g., "Otoño", "Primavera").
  • ❌ No se permiten valores vacíos ni tipos erróneos.
