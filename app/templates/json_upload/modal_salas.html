🏫 Subir Salas de Clase

Sube el archivo 8-salas_de_clases.json con la información de las salas disponibles en el sistema.
El archivo debe tener la siguiente estructura: 

{
    "salas": [
      {
        "nombre": "Sala 101",
        "capacidad": 40
      },
      {
        "nombre": "Laboratorio A",
        "capacidad": 25
      }
    ]
  }
Validaciones aplicadas:
  • ✔️ nombre debe ser un string no vacío.
  • ✔️ capacidad debe ser un entero mayor a 0.
  • ❌ No se permiten nombres duplicados.
