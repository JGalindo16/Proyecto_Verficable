📥 Subir Inscripciones de Alumnos

Sube el archivo 6-alumnos_por_seccion.json con la información de inscripción de estudiantes a secciones. Asegúrate de que el archivo tenga una estructura como: 

{
    "inscripciones": [
      {
        "student_id": 1,
        "section_id": 101
      }
    ]
  }
Validaciones que se aplicarán:
  • ✔️ student_id debe ser un número entero y existir en la tabla students.
  • ✔️ section_id debe ser un número entero y existir en la tabla sections.
  • ❌ No se permitirán inscripciones duplicadas.
  • ❌ No se permitirá inscripción si alguno de los IDs referenciados no existe.
